# Supabase Email Templates

These templates are configured in the Supabase Dashboard under:
**Authentication > Email Templates**

## SMTP Setup (Resend)

Configure in **Authentication > SMTP Settings**:
- Host: `smtp.resend.com`
- Port: `465`
- Username: `resend`
- Password: Your Resend API key (`re_xxxxxxxxxxxx`)
- Sender email: `noreply@dealsletter.io`
- Sender name: `Dealsletter`

## Templates

Copy-paste each template below into the corresponding tab in the Supabase dashboard.
